# Cyfig

An online cyberware configurator for *Cyberpunk 2077*. Plan a build by slotting cyberware onto a character silhouette while the app tracks your cyberware capacity and factors in attribute points, so you can see exactly what fits and which buffs you'll get before committing in-game.

## Features

- Cyberware capacity meter that updates as you equip parts
- Attribute point selector that factors stat-based perks and capacity into the build
- Per-slot cyberware selection on a human silhouette
- Sidebar listing all active cyberware buffs
- Edgerunner perk toggle
- All cyberware available at max tier

## Tech stack

- **Framework:** React 18 + TypeScript
- **Build tool:** Vite
- **Styling:** Tailwind CSS
- **Icons:** lucide-react

## Setup

```bash
npm install
npm run dev          # start the dev server (Vite prints the local URL)
npm run build        # production build to dist/
npm run preview      # preview the production build
```
